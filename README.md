# 🚀 REST API com .NET 5

Esta é uma API RESTful robusta e escalável desenvolvida com **.NET 5** (ASP.NET Core Web API). O projeto foi estruturado seguindo as melhores práticas do desenvolvimento backend, com foco na separação de responsabilidades, injeção de dependência e manipulação correta do protocolo HTTP.

---

## 🎯 Principais Recursos e Práticas Adotadas

* **Design RESTful:** Uso correto dos verbos HTTP (`GET`, `POST`, `PUT`, `DELETE`) e retorno de status codes apropriados (`200 OK`, `201 Created`, `404 Not Found`, etc.).
* **Injeção de Dependência:** Utilização do container nativo do ASP.NET Core para gerenciamento do ciclo de vida de serviços e repositórios.
* **Padrão Repository (Repository Pattern):** Camada de abstração de dados para isolar a lógica de negócio do acesso direto ao banco de dados ou coleções.
* **Validação de Modelos:** Uso de Data Annotations para garantir a integridade dos dados recebidos nas requisições antes do processamento.
* **Documentação com Swagger:** Integração com OpenAPI (Swagger) para facilitar o teste dos endpoints diretamente pelo navegador.

---

## 🛠️ Tecnologias Utilizadas

* **Framework Principal:** .NET 5 (ASP.NET Core Web API)
* **Ferramenta de Teste/Documentação:** Swagger / OpenAPI
* **Linguagem:** C#
